# 06 · AI 分析

本章说明如何配置模型、圈定数据范围、发起分析对话，以及如何把结果落到控制台、画布与外部工单。

---

## 6.1 功能说明

AI 分析工作台把「提问 → 生成 SQL → 执行 → 图表/报告」串成流水线，并可保存为可参数化重跑的 **分析画布**。

![AI 分析工作台](../assets/screenshots/04-ai-analysis.png)

**图中信息：** 中央为对话与分析步骤（含 SQL 预览与结果）；可切换模型；侧栏选择已 AI 打标的表；完成后可「在控制台打开」或「保存为画布」。

---

## 6.2 使用前检查清单

按顺序确认，缺一不可：

| # | 检查项 | 在哪里做 |
|---|--------|----------|
| 1 | AI Provider / API Key / 默认模型 | **设置 → AI 模型**（截图 `09-settings-ai.png`） |
| 2 | 表已 AI 打标 | 库 → **查看所有表** → 标记（第 5.9 节） |
| 3 | 账号有 AI / 分析权限 | 管理员在权限中心配置 |
| 4 | （可选）DataAgent 流水线参数 | 设置中 DataAgent 分区 |

未打标时：侧栏数据范围为空，分析可能退化为普通聊天，或无法生成带执行的流水线。

---

## 6.3 打开 AI 分析

| 入口 | 路径 |
|------|------|
| 顶栏导航 | **AI 聊天** / AI |
| 仪表盘 | **AI 数据分析** 快捷操作 |
| 命令面板 | `Ctrl+K` 搜「AI」 |
| 资源树 | 部分表操作也可跳到 AI（视菜单） |

---

## 6.4 第一次完整对话（推荐步骤）

1. 打开 AI 分析页。
2. 在顶部或侧栏 **选择模型**（与设置中的默认一致亦可）。
3. 打开 **数据范围**，勾选已打标的表（可多选；跨库时结果可能带 `__dw_source__` 来源列）。
4. 在输入框用自然语言描述问题，例如：  
   - 「最近 7 天订单金额按天汇总」  
   - 「找出从未下单的用户」  
5. 按 `Enter` 或点击发送。
6. 等待流水线：理解问题 → 生成 SQL → 执行 → 可视化 / 文字结论。
7. 审阅 SQL：是否扫了过大表、是否缺过滤条件、是否写操作（一般分析应为只读）。

### 提问技巧

- 写明时间范围、指标定义、分组维度。
- 指明表名（若已打标多张表）。
- 一次问一个清晰目标，再追问细化。

---

## 6.5 结果区能做什么

| 操作 | 何时用 | 下一步 |
|------|--------|--------|
| **在控制台打开** | 要改 SQL、加 JOIN、手工调优 | 进入第 4 章控制台 |
| **保存为画布** | 要参数化重跑、给同事复用、挂定时 | 资源树 **AI → 分析画布**（第 8.2） |
| 导出 Markdown / HTML | 写周报、存档 | 下载后外发注意脱敏 |
| 导出工单 | 要研发跟进 | 需配置 GitHub / GitLab / Jira 集成 |

---

## 6.6 与「控制台内 AI」的区别

| | AI 分析工作台（本章） | 控制台 Text-to-SQL |
|--|----------------------|-------------------|
| 定位 | 端到端分析 + 图表 + 画布 | 辅助写 SQL |
| 数据范围 | 显式勾选已打标表 | 依赖当前控制台上下文 |
| 输出 | 对话、步骤、可视化、报告 | 主要是 SQL 文本 |
| 适合 | 业务问题、探索分析 | 已有控制台、快速补语句 |

---

## 6.7 配额与限流

- 租户可设 **日限额**；接近或用尽时界面提示。
- 管理员在租户/组织设置中调整配额。
- 失败时区分：配额用尽 vs API Key 无效 vs 网络错误。

---

## 6.8 安全注意

- 只把必要的表打标并勾选进范围。
- 不要把含明文密钥的表结构/样例发给不可信模型端点。
- 导出报告前检查是否含 PII；启用脱敏策略时确认仍生效。
- 生产库优先只读账号给 AI 使用。

---

## 6.9 常见问题

| 现象 | 处理 |
|------|------|
| 没有数据范围可选 | 先 AI 打标（5.9） |
| 一直失败 / 无流式输出 | 检查 API Key、模型名、后端 AI 代理日志 |
| SQL 跑错库 | 打标表来自哪条连接；范围是否勾错 |
| 想定时每天跑 | 保存画布 → **定时任务** 选画布类型（第 8 章） |
| 想改 SQL 再跑 | 「在控制台打开」 |

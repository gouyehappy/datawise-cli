# 02 · 界面总览

认识主界面分区、导航、命令面板、Tab 与常用快捷键，方便后续章节「按入口找功能」。

## 2.2 左侧导航怎么用

| 入口 | 进入后 | 典型下一步 |
|------|--------|------------|
| **数据库** | 工作台：左侧资源树 + 中央 Tab | 建连接、开 SQL、开表（第 3–5 章） |
| **仪表盘** | 概览与快捷操作 | 看健康、新建控制台、进 AI（第 7 章） |
| **AI** | AI 分析全屏工作台 | 选范围、提问（第 6 章） |
| **插件** | 插件中心 | 启停插件、装连接器（第 11 章） |

### 注意

- 已在「数据库」且资源树展开时，再点一次 **数据库** 会 **收起** 资源树。
- 桌面版顶栏可能有 **配置 / 帮助** 下拉：布局开关、导览、关于。

---

## 2.3 命令面板（全局最快入口）

**打开：** `Ctrl+K`（可在设置 → 快捷键修改）。

**能搜到什么：**

- 模块跳转（仪表盘、设置、插件、联邦…）
- 书签 / 最近 SQL 编辑器
- 插件启停类命令
- 组织数据目录相关入口

**建议习惯：** 忘记菜单在哪时，先 `Ctrl+K` 输入中文关键词。

截图叠加效果见 `02-explorer.png`。

---

## 2.4 全局对象搜索

**打开：** 顶栏「找对象」或 `Ctrl+Shift+O`。

按连接 / 库 / 表定位，适合树很深、记不清路径时。选中后跳到资源树或打开对象。

---

## 2.5 工作区 Tab

中央可同时打开多种 Tab，例如：

| Tab 类型 | 来源示例 |
|----------|----------|
| SQL 控制台 / SQL 编辑器 | 双击库、右键、仪表盘新建 |
| 表数据 | 双击表 |
| 连接编辑表单 | 编辑连接 |
| 平台目录 | AI → 画布 / 联邦 / 漂移… |
| Redis / Kafka / Yarn / SSH | 对应连接器节点 |
| 数据目录 | 命令面板打开 |

切换 Tab 不影响资源树选中态；可用树工具栏 **定位** 反查当前 Tab 对应节点。

---

## 2.6 右侧工具栏

常见入口（图标因布局设置可能显隐）：

- 信息 / 文档类
- SQL 历史
- 系统监控
- 书签（Query Library）
- Schema 迁移相关快捷入口
- 导出相关

显隐与预览：见 **设置 → 界面布局**（`07-settings-layout.png`）。

---

## 2.7 通知抽屉

**打开：** 左侧底部通知图标，或 `Ctrl+Shift+N`。

常见通知：连接健康变化、慢查询、定时任务失败、审批待办、漂移告警等。

规则配置：**设置 → 连接健康**（第 11 章）。

---

## 2.8 常用快捷键（默认）

| 操作 | 典型绑定 |
|------|----------|
| 命令面板 | `Ctrl+K` |
| 全局对象搜索 | `Ctrl+Shift+O` |
| 打开设置 | `Ctrl+,` |
| 通知 | `Ctrl+Shift+N` |
| 执行 SQL | 常见 `Ctrl+Enter` 类（以快捷键页为准） |
| 控制台 AI | `Alt+/` |
| 资源树内搜索 | `Ctrl+F`（焦点在树上） |

修改：`Ctrl+,` → **快捷键** → 点击绑定区 → 按下新组合键。

---

## 2.9 主题与布局

- 界面款式 / 主题：设置 → 基础设置（`06-settings-basic.png`）
- 导航栏、工具栏、右侧栏显隐：设置 → 界面布局（`07-settings-layout.png`）

改完可点底部 **返回工作台** 立即查看效果。
